About EventHub 🚀

🎯 What is EventHub?

EventHub is a modern event registration and management platform designed to simplify how users discover, register, and manage events. It provides a seamless experience for both users and administrators with features like OTP-based authentication, QR-based ticketing, and real-time event tracking.

⚙️ How it Works?

  • User signs up using Email OTP verification
  • Browse available events (Free & Paid)
  • Register for events easily
  • Receive a unique QR ticket
  • Scan QR at entry for verification

💡 Why EventHub?

It solves real-world problems like manual registrations, fake entries, and lack of tracking. With a clean UI and secure backend, EventHub ensures a smooth and professional experience.

✨ Key Features

  • 🔐 OTP Authentication
  • 🎟 QR Ticket System
  • 💳 Payment Flow
  • 📧 Email Notifications
  • 📱 Mobile Friendly UI
  • 🛠 Admin Dashboard
